‚óè  Why is React known as React? 

React is named react because of its main concept it reacts to changes in data.
the core concept behind react is the concept of virtual DOM
when data changes in react application instead of directly manipulating the DOM, react creates a virtual 
representaion of DOM in memory, compares it with the previous state,and then calculates the most efficient way to update the DOM
The name react refelects the reactive nature of the framework

‚óè  What is crossorigin in script tag? 
The crossorigin attribute in a script tag is used to specify how the browser should handle requests
made to load the script from a different origin  than the one hosting the current web page.
it is used for security purpose

3 possible cross origin attribute
anonymous--default value  crossorigin="anonymous"

use credentials  

3.empty string or absent--it will handle the request according to the CORS
